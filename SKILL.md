# 即时分析

按本目录 `PLAYBOOK.md` 执行，它是唯一的行为规范（路由表、材料规范、模板路由、
诚实守则、降级话术都在里面）。本文件只是 Claude Code 的触发入口。

要点提醒：

1. 先读 `PLAYBOOK.md`，按输入类型走路由表抓取，脚本在 `scripts/`（零依赖，Node 18+）
2. 解读模板在 `templates/`，按内容类型自动选，用户一句话可覆盖；
   用户自定义模板 = 目录里的新 .md 文件
3. 诚实守则优先级最高：拿不到的内容如实降级并显式声明，不脑补
4. ASR 转写是可选增强（需 faster-whisper），转写前告知用户需等几分钟
